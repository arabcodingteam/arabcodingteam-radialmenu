## Features
- House Decor
- Actions
- Walking Style Menu
- Vehicle Control
- Vehicle Extra Menu

## Installation
### Manual
- Download the script and put it in the `[qb]` directory.
- Add the following code to your server.cfg/resouces.cfg
```
ensure qb-core
ensure qb-radialmenu
ensure qb-menu
ensure vehcontrol
```
